# 🌱 Erica's 작물 키우기 게임

5가지 작물을 키우는 파이썬 pygame 게임입니다.

## 작물 종류
- 감자
- 상추
- 해바라기
- 완두콩
- 사과나무

## 실행 방법
```bash
pip install -r requirements.txt
python src/main.py
```
## 오류
-- 작물이 화분크기에 딱 맡게 자람(화분을 못넘어감)
